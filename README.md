## Информация:
В даном репозитории находиться реализация Backend для сайта ресторана/кафе.
## Технологии:
JWT, SQLAlchemy, FastAPI, PyDantic, AuthX, Bcrypt, Postgesql
## Как запустить:
Нужно установить Postgresql, Python 3.12.6.
Также нужно создать базу данных localhost
Всё что можно настроить находиться в .env
```bash
uv sync
cd src/
uv run main.py
```
